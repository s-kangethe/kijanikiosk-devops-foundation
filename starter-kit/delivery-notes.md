How Flow, Feedback and Learning is demonstrated in my workflow:
Flow: I have used the step by step instructions to create the required branches and their files and named them as required
thus creating the structure of the repository.I then proceeded to work on the tasks for each file/branch, creating my flow from structure to tasks.
Feedback: I obtain feedback based on responses after inputing commands on the terminal. I have used commands like git branch -a to check if all my branches are there, ls to check the list of files in a directory etc.I have used feedback to check the correctness of my command actions.
Learning: By figuring out how to create folders, files, branches etc, I learn from doing and figuring out how to fix the bugs I experience.
